# Vincent Van Git
Use your Github commit history as a canvas!

### Install

#### Clone this repo

```
git clone https://github.com/jh3y/electron-parcel-react-starter.git
```

#### Install dependencies

```
npm install
```

or

```
yarn
```

### Usage

#### Run the app

```
npm run dev
```

or

```
yarn dev
```

#### Build the app (automatic)

```
npm run package
```

or

```
yarn package
```
